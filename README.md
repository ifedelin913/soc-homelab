# SOC / SIEM Home Lab

A hands-on security operations lab environment built to practice and demonstrate the relevant skills and workflows of detection engineering, incident response, and threat intelligence.

## Architecture

## Skills Demonstrated
| Skill | Tool |
|---|---|
| SIEM administration | Elastic Stack |
| Log ingestion & normalization | Filebeat, Winlogbeat |
| Detection rule writing | KQL, Sigma |
| Network traffic analysis | Zeek, Suricata |
| Incident response | TheHive, custom playbooks |
| Threat intelligence | MISP, AlienVault OTX |
| Adversary simulation | Kali Linux, Metasploit |
